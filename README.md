# python-mini-projects
Collection of Python mini projects including file handling, web scraping, API integration, encryption, and algorithms.
This repository contains beginner Python projects built to practice
core programming concepts.

Projects included:
- Word Counter (file handling)
- Web Scraper (BeautifulSoup)
- API Integration (CoinGecko)
- File Encryption/Decryption
- N-Queens Algorithm

Each project runs independently using:
python filename.py
